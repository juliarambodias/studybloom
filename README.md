# StudyBloom 🌱

Plataforma de estudos para organizar aulas, áudios e gerar resumos automaticamente.

## 🚀 Funcionalidades
- Upload de aulas (áudio/vídeo)
- Reprodução direta no navegador
- Transcrição automática (limitada ao navegador)
- Geração de resumo
- Organização por aulas

## 🧠 Objetivo
Este projeto foi criado para ajudar nos estudos da faculdade, permitindo armazenar aulas e revisar conteúdos de forma prática.

## ⚠️ Limitações atuais
- Transcrição não funciona perfeitamente para áudios longos
- Dados salvos apenas no navegador (localStorage)
- Sem backend (ainda)

## 🔮 Próximos passos
- Integração com IA para transcrição completa
- Armazenamento em nuvem
- Melhorias na geração de resumos
